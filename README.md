# 통장 거래내역 변환기

은행 거래내역 HTML 파일을 엑셀(xlsx)로 변환하는 데스크톱 애플리케이션입니다.

Go + [Fyne](https://fyne.io/) GUI 프레임워크로 개발되었습니다.

## 다운로드

[Releases](https://github.com/yonsei-alumi/bank-sheet/releases/latest) 페이지에서 운영체제에 맞는 파일을 받으세요.

| 파일 | 운영체제 |
|------|----------|
| `bank-sheet-linux-amd64` | Linux (x86_64) |
| `bank-sheet-windows-amd64.exe` | Windows (x86_64) |

## 지원 은행

- 신한은행 모임통장

## 사용법

### HTML 파일 준비

신한은행 모바일 웹에서 모임통장 거래내역 페이지를 열고, 브라우저의 "페이지 저장" 기능으로 HTML 파일을 저장합니다.

### 엑셀 변환

1. 프로그램을 실행합니다.
2. 은행 유형을 선택합니다. (현재 "신한은행 모임통장" 지원)
3. "HTML 파일 선택" 버튼을 클릭하여 저장한 HTML 파일을 선택합니다.
4. 거래 내역이 분석되고, 같은 디렉터리에 동일한 이름의 엑셀 파일(.xlsx)이 자동 생성됩니다.
5. 생성 완료 후 "생성된 파일을 열까요?" 대화상자에서 Yes를 누르면 바로 열 수 있습니다.

### 엑셀 파일 형식

| 열 | 내용 | 형식 |
|----|------|------|
| 입금일자 | 거래 일시 | YYYY-MM-DD HH:mm:ss |
| 입금자 | 입금한 사람 이름 | 문자 |
| 입금액 | 입금 금액 | 숫자 (천 단위 콤마) |
| 잔액 | 거래 후 잔액 | 숫자 (천 단위 콤마) |

### 파일명 규칙

- HTML 파일과 동일한 이름에 확장자만 `.xlsx`로 변경됩니다.
- 같은 이름의 파일이 이미 있으면 `이름(1).xlsx`, `이름(2).xlsx` 순으로 생성됩니다.

## 요구사항

- Go 1.24+
- CGO 지원 (Fyne 빌드에 필요)

## 빌드

```bash
# Linux + Windows 동시 빌드
make

# Linux만
make linux

# Windows만 (크로스 컴파일, x86_64-w64-mingw32-gcc 필요)
make windows
```

빌드 결과물은 `dist/` 디렉터리에 생성됩니다.

## 실행

```bash
# 소스에서 직접 실행
go run .

# 빌드된 바이너리 실행
./dist/bank-sheet-linux-amd64
```
